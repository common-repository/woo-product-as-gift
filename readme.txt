== Description ==

This plugin gives you the ability to add gift wrapping on your order.

== Installation ==
Important: First of all, you have to download and activate WooCommerce plugin, because WC Gift Cards cannot work without it .

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the wp-content/plugins/ directory of your WordPress site.
3. Activate from Plugins page.

Voila! you can add gift wrap now.

== Admin Settings ==

1- Go to WooCommerce tab and click on Gift Settings from the list.
2- There you can see two tabs 'Wrapping' and 'Note'.
3- On 'Wrapping' tab, click on the checkbox to Enable Gift Options.
4- Admin can provide the Label by own choice.
5- Set the wrapping style title, price and upload any image.
6- To apply the settings click on the Apply button.
7- On 'Note' tab, click on the checkbox to Charge Fee for message.
8- Admin can add charges for message accodringly.
9- Admin can add the Label for Add a message with gift field.
10- To apply the settings, hit the Apply button after changes.
